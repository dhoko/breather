## Install

```
$ git clone git@github.com:kud/breather.git
```

```
$ make install
```

## Settings

```javascript
{
  "duration": "25", // duration of a session
  "location": "Paris,fr", // where you are
  "lang": "en" // language
}
```

## Usage

```
$ make start
```
